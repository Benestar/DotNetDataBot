DotNetDataBot

Extension of DotNetWikiBot framework for Wikibase extension of Mediawiki

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the documentatoin of DotNetDataBot

DotNetDataBot is an extension of DotNetWikiBot to fit to Wikibase, an extension of MediaWiki. It is written in C# and runs on .NET Framework 4.0, perhaps even lower.

Download

The source code will be published soon but you can download the compiled documents before. Click here to get the package. After unpacking it you can see a file called DotNetDataBot.dll and one called DotNetDataBot.xml. The xml document is only for documentation. To use it you have to create a new refer in your project. Then you can write using DotNetDataBot; to import the framework.

Example

Here is a short example how you can use DotNetData framework:

using DotNetDataBot;

namespace MyBot
{
    class Example
    {
        public static void Main(string[] args)
        {
            Site site = new Site("http://www.wikidata.org", "username", "password"); // replace username and password with you bot's login data
            Item item = new Item(site, "Q1000");
            item.setSiteLink("en", "Earth"); // sets the sitelink

            Item newItem = new Item(site);
            newItem.createItem("en", "Wikidata", "database for collecting interwikilinks"); // creates a new item
        }
    }
}
